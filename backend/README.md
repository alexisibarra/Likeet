# Likeet Backend

## Requirements

- Python
- Create a virtualenv `likeet` (Example name)
- Mongodb

## Install

    $ git clone git@github.com:alexisibarra/Likeet.git
    $ cd Likeet/backend
    $ pyenv activate likeet
    $ pip install flask_mongoengine flask_script flask_cors

## To Run

    $ cd Likeet/backend
    $ python app.py
